To style a checkbox with CSS, you can hide the default checkbox and create a custom design using labels and pseudo-elements. Here's a simple example:

✅ HTML

html
<label class="custom-checkbox">
  <input type="checkbox" id="myCheck">
  <span class="checkmark"></span>
  Accept Terms
</label>

🎨 CSS

css
/* Hide the default checkbox */
.custom-checkbox input {
  display: none;
}

/* Style the custom checkbox box */
.custom-checkbox .checkmark {
  height: 20px;
  width: 20px;
  background-color: #eee;
  border-radius: 4px;
  display: inline-block;
  vertical-align: middle;
  margin-right: 10px;
  position: relative;
  transition: 0.2s;
}

/* Checked state */
.custom-checkbox input:checked + .checkmark {
  background-color: #2196F3;
}

/* Add a checkmark symbol */
.custom-checkbox .checkmark::after {
  content: "";
  position: absolute;
  display: none;
  left: 6px;
  top: 2px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 3px 3px 0;
  transform: rotate(45deg);
}

/* Show the checkmark when checked */
.custom-checkbox input:checked + .checkmark::after {
  display: block;
}

Let me know if you want it styled in a specific theme (e.g., dark mode, round checkbox, animated, etc.)!

Great! If you want users to see the checkbox, but change its background and color when checked, here’s a simple way to do it without hiding the checkbox:

✅ HTML

html
<input type="checkbox" id="myCheck">
<label for="myCheck">Accept Terms</label>

🎨 CSS

css
/* Style the checkbox */
input[type="checkbox"] {
  width: 20px;
  height: 20px;
  accent-color: #2196F3; /* Changes checkmark color in supported browsers */
  background-color: white;
  border: 2px solid #ccc;
  border-radius: 4px;
  cursor: pointer;
}

/* When the checkbox is checked */
input[type="checkbox"]:checked {
  background-color: #2196F3; /* Background changes when checked */
  border-color: #2196F3;
}

✅ Notes:

  • accent-color works in modern browsers to change the tick color.

  • You can remove accent-color if not supported and use a full custom checkbox instead.

Yes, you can change the color of the tick (✔️) using the CSS accent-color property — but it's only supported in modern browsers like Chrome, Edge, Firefox, and Safari.

✅ Example: Change Checkbox Tick Color

html
<input type="checkbox" id="myCheck">
<label for="myCheck">Accept Terms</label>

css
input[type="checkbox"] {
  width: 20px;
  height: 20px;
  accent-color: red; /* This changes the tick color */
}

🎯 Supported Colors:

  • accent-color changes the tick mark color and the background when checked.

  • If you set accent-color: red;, the checkbox tick becomes red when selected.

⚠️ Browser Support

accent-color is supported in:

  • ✅ Chrome 93+

  • ✅ Edge 93+

  • ✅ Firefox 92+

  • ✅ Safari 15.4+

If you're building for older browsers or want full control, you'd need to hide the native checkbox and build your own with CSS, like the first example I shared.
